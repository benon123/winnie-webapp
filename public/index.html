About This Project

This webpage forms part of a coursework project designed to demonstrate the development of a simple web application using modern web technologies.

Project Description

The objective of this project is to design and implement a basic web application that demonstrates server-side development using Node.js and Express.js. The application includes multiple web pages, a structured routing system, and a contact form capable of sending user data to the server.

Technologies Used

  • Node.js runtime environment
  • Express.js framework for server-side routing
  • EJS template engine for dynamic page rendering
  • Bootstrap 5 for responsive user interface design
  • HTML5 and CSS3 for webpage structure and styling

Learning Outcomes

Through the development of this project, key skills were acquired in structuring a web application, implementing middleware, handling HTTP requests, and applying responsive design principles. The project also demonstrates how backend and frontend components integrate to create functional web systems.
